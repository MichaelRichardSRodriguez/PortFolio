Technologies Used in the Web Application

ASP.Net MVC

ASP.Net MVC is used for building web applications following the Model-View-Controller pattern. It separates logic, UI, and data to enhance maintainability.

Entity Framework

Entity Framework (EF) is an ORM (Object Relational Mapper) used to interact with the SQL Server database without writing raw SQL queries, streamlining data operations.

SQL Server

SQL Server is the relational database used for storing and managing data in this web application. It provides secure, reliable, and efficient data management.

Razor View

Razor Views are used for generating dynamic HTML content on the server side with clean syntax and easy integration with C#.

Razor Page

Razor Pages are used for creating web pages in an easy and efficient manner by combining page logic and UI in a single file.

Identity

ASP.Net Identity is used for managing user authentication and authorization, providing features like registration, login, and role management.

Bootstrap

Bootstrap is a powerful front-end framework that provides pre-built styles, components, and JavaScript plugins to build responsive, mobile-first websites.

Ajax

Ajax (Asynchronous JavaScript and XML) is used to perform asynchronous requests to the server, allowing content to be updated without refreshing the entire page.

JavaScript

JavaScript is used to add interactivity to the web application, including event handling, dynamic content updates, and client-side validation.
